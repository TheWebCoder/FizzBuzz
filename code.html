                        //get the values from the page
                        //start or controller function
                        function getValues() {
                            //get values from the page
                            let fizzValue = document.getElementById("fizzValue").value;
                            let buzzValue = document.getElementById("buzzValue").value;
                            

                            //We need to validate our input
                            //parse into Integers
                            fizzValue = parseInt(fizzValue);
                            buzzValue = parseInt(buzzValue);

                            if (Number.isInteger(fizzValue) && Number.isInteger(buzzValue)) {
                                //call fizzBuzz
                                let fbArray = fizzBuzzC(fizzValue,buzzValue);
                                //call display data by passing in the array
                                displayData(fbArray);
                            } else {
                                alert("You must enter integers");
                            }
                        }


                        //Initiate FizzBuzz
                        function fizzBuzz(fizzValue,buzzValue) {

                            let returnArray = []

                            //loop from 1 to 100
                            //Check to see if divisible by fizz value, buzz value, or both
                            for (let i = 1; i <= 100 ; i++) {
                                if (i % fizzValue == 0 && i % buzzValue == 0) {
                                    returnArray.push("FizzBuzz")
                                } else if (i % fizzValue == 0) {
                                    returnArray.push("Fizz");
                                } else if (i % buzzValue == 0) {
                                    returnArray.push("Buzz");
                                } else {
                                    returnArray.push(i);
                                }
                            }
                            return returnArray;
                        }

                        function fizzBuzzB(fizzValue, buzzValue) {
                            let returnArray = [];
                            let Fizz = false;
                            let Buzz = false;

                            for (let i = 1; i <= 100; i++) {

                                    Fizz = i % fizzValue == 0;
                                    Buzz = i % buzzValue == 0;

                                    switch(true)
                                    {
                                        case Fizz && Buzz:{
                                            returnArray.push("FizzBuzz");
                                            break;
                                        }
                                        case Fizz: {
                                            returnArray.push("Fizz");
                                        }
                                        case Buzz: {
                                            returnArray.push("Buzz");
                                        }
                                        default:{
                                            returnArray.push(i);
                                            break;
                                        }
                                    }
                            }

                            return returnArray;
                        }

                        function fizzBuzzC(fizzValue, buzzValue) {
                            let returnArray = [];
                        

                            for(let i = 1; i <= 100; i++) {

                                let value = ((i % fizzValue == 0 ? "Fizz" : "") + (i % buzzValue == 0 ? "Buzz" : "" ) || i);
                                returnArray.push(value);

                            }

                            return returnArray;
                        }

                        function displayData(fbArray) {

                            //get the table body element from the page
                            let tableBody = document.getElementById("results");

                            //get the template row
                            let templateRow = document.getElementById("fbTemplate");

                            //clear table first
                            tableBody.innerHTML = "";

                            for (let i = 0; i < fbArray.length; i += 5) {

                                let tableRow = document.importNode(templateRow.content, true);

                                //grab just the td and put into array
                                let rowCols = tableRow.querySelectorAll("td");
                                rowCols[0].classList.add(fbArray[i]);
                                rowCols[0].textContent = fbArray[i];

                                rowCols[1].classList.add(fbArray[i + 1]);
                                rowCols[1].textContent = fbArray[i+1];

                                rowCols[2].classList.add(fbArray[i + 2]);
                                rowCols[2].textContent = fbArray[i+2];

                                rowCols[3].classList.add(fbArray[i + 3]);
                                rowCols[3].textContent = fbArray[i+3];

                                rowCols[4].classList.add(fbArray[i + 4]);
                                rowCols[4].textContent = fbArray[i+4];

                                tableBody.appendChild(tableRow);

                            }

                        }

The Code is structured in five functions, three of them are different implementations of the FizzBuzz challenge. fizzBuzzC is the main and most efficient implementation.

getValues()

This function is the starter or controller of the program. It activates when the submit button is clicked. The function fetches 'fizzValue' and 'buzzValue' from the user input, converts them to integers, and checks if they are valid. If both are integers, it calls 'fizzBuzzC' with these values and receives an array back. This array is then given to 'displayData' for showing on the page. If either input is not an integer, an alert pops up to inform the user.

fizzBuzz(fizzValue, buzzValue)

As a logic function, it runs a for loop from 1 to 100. It checks each number against 'fizzValue' and 'buzzValue'. Depending on divisibility, it adds "Fizz", "Buzz", "FizzBuzz", or the number itself into an array. The array is returned after the loop finishes.

fizzBuzzB(fizzValue, buzzValue)

Another logic function for FizzBuzz using a switch statement, creating an array from 1 to 100. It uses boolean checks 'Fizz' and 'Buzz' for divisibility and a switch statement to decide what goes into the array: "Fizz", "Buzz", "FizzBuzz", or the number. The array is returned at the end.

fizzBuzzC(fizzValue, buzzValue)

This is the main version of FizzBuzz which also loops from 1 to 100. It combines "Fizz", "Buzz", or both in a string, or keeps the number, based on divisibility by 'fizzValue' and 'buzzValue'. The results form an array, which is then returned.

displayData(fbArray)

This function displays the FizzBuzz array on the web page. It begins by selecting the table body and a template row, then clears the table. It fills the table with rows of data from the array, displaying five results per row. The function uses 'document.importNode' for cloning the template for each row, and 'classList.add' to add classes to the table data as per the FizzBuzz results.
